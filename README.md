# 钢铁侠战甲工坊 · Iron Man 3D Workshop

浏览器里的钢铁侠战甲交互展示：MK85 战甲、81 件装甲分区拆解（头部/躯干/双臂/双腿）、爆炸图滑杆、穿甲动画、双击聚焦细节、灯光开关、贾维斯风格 HUD。

在线体验：https://rose-for-future.github.io/ironman-3d/

Three.js 构建，纯静态零后端。
